# Little Free Pantry Initiative

**[brooksgroves.com/little-free-pantry](https://brooksgroves.com/little-free-pantry/)**

Interactive map and project page for the [Lakewood Rotary](https://lakewoodrotary.com) Little Free Pantry program.

## About

During the 2020 pandemic, Lakewood Rotary launched the Little Free Pantry project to provide 24/7 barrier-free access to food and hygiene items across Lakewood, Tillicum, and Steilacoom, WA. Currently 7 pantries across Lakewood, Tillicum, and Steilacoom, supported by 5 community drop-off sites.

## Features

- **Leaflet interactive map** — Rotary-branded markers, rich popups with directions links
- **Photo gallery** — actual pantry photos showing different cabinet styles and signage
- **Project story** — background, community impact, food desert context
- **Current needs** — most-needed items list
- **QR code** — scan-to-donate integration
- **Original PDF map** — collapsible view of the December 2025 reference map

## Tech

Plain HTML/CSS/JS. No frameworks, no build step.

- [Leaflet](https://leafletjs.com/) with CARTO Voyager tiles
- Google Fonts (Playfair Display, Source Sans 3)
- GitHub Pages hosting
